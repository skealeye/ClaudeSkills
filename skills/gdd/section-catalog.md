# GDD Section Catalog

## Default Sections

| # | Section | When Applicable | Suggested H2 Headings |
|---|---------|-----------------|----------------------|
| 01 | Game Overview | Always | Design Pillars, Elevator Pitch, Target Audience, Comparable Titles, Unique Selling Points, What Success Looks Like |
| 02 | World & Lore | Always | Setting Overview, Factions, Key Locations, Narrative Tone, Lore Delivery Methods |
| 03 | Core Loop | Always | Primary Loop, Session Flow, Nested Loops, Tension Curve, Decision Points, One-More-Run Psychology |
| 04 | Exploration | Games with traversal/discovery | Movement Mechanics, Discovery Systems, Door & Lock Systems, Environmental Storytelling, Points of Interest |
| 05 | Combat System | Games with combat | Combat Philosophy, Attack/Defense Mechanics, Enemy Types, Boss Design, Damage Model, Reaction Systems |
| 06 | Card System | Games with card mechanics | Card Anatomy, Card Types & Rarities, Deck Building, Card Catalog, Synergies & Archetypes, Expansion Framework |
| 07 | Zones & Maps | Games with distinct areas | Zone Overview, Per-Zone Breakdown (setting, layout, enemies, hazards, boss), Zone Progression, Procedural Generation Rules |
| 08 | Extraction Mechanics | Extraction-style games | Extraction Philosophy, Loot & Carry System, Risk/Reward Tension, Beacon/Exit Mechanics, Loss Conditions, Timer Systems |
| 09 | PvP System | Games with PvP | PvP Philosophy, Matchmaking, Combat Rules, Anti-Grief Measures, Ranking System, Rewards |
| 10 | Progression & Economy | Games with progression/currency | XP & Leveling, Unlock Tree, Currencies, Crafting, Economy Balance, Prestige Systems |
| 11 | The Hideout | Games with a hub/base | Hideout Philosophy, Room Types, Upgrade Paths, NPC Interactions, Stash Management, Social Features |
| 12 | UI-UX | Always | HUD Design, Menu Flow, Accessibility, Controller Support, Notification Systems, Social UI |
| 13 | Art Direction | Always | Visual Style, Color Palette, Character Design Language, Environment Art, VFX Philosophy, Performance Targets |
| 14 | Audio Design | Always | Audio Philosophy, SFX Design, Dynamic Music, Spatial Audio, Announcer/VO, Adaptive Mixing |
| 15 | Monetization | Games with monetization | Business Model, Cosmetics Strategy, Season Pass, Shop Design, Ethical Monetization Rules, Revenue Projections |
| 16 | Development Roadmap | Always | Phase Breakdown, Milestone Definitions, Risk Register, Resource/Budget Estimates, Tooling |

## Content Type Guidance

| Content Type | When to Use | Format |
|---|---|---|
| Tables | Stats, parameters, item lists, comparisons, loot tables | Markdown tables |
| ASCII Diagrams | Flows, spatial layouts, state machines, HUD mockups | Code-fenced blocks |
| Prose | Philosophy, narrative, world-building | Paragraphs with bold/italic emphasis |
| Callout Boxes | Design rules, warnings, tips, cross-references | `> [!TYPE]` Obsidian syntax |
